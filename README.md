No Sistema Thomaz Ferrovia, todos os trabalhadores devem possuir cadastro
com usuário e senha. O acesso ao sistema é seguro, com possibilidade
de redefinição de senha via e-mail e código de verificação. Após o login, o usuário visualiza
o dashboard, que permite navegação por menu, possíveis rotas, acesso a alertas em tempo real, relatórios
com gráficos de consumo de energia e eficiência operacional, além do monitoramento dos trens.


REGRAS DE NEGÓCIO:

RN001 Todos os trabalhadores precisam de um cadastro com usuário e senha.
RN002 Todos os trabalhadores devem receber avisos no app.
RN003 Relatórios devem conter gráficos de consumo de energia e eficiência operacional.
RN004 Qualquer trabalhador pode acionar a equipe de manutenção ou notificar o motorista.
RN005 Todo funcionário deve possuir um número de registro.
RN006 O sistema deve permitir redefinição de senha via e-mail e código de verificação.
RN007 Trens operam até 200 km/h para estimar chegada.
RN008 Toda viagem tem origem e destino definidos.
